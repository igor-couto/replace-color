# replace-color
Rust CLI application to replace a color


## Build

Compile with
```
cargo build --release
```

## Usage

Put the replace-color.exe file in your PATH.
Replace all red pixels of image.png from blue ones:

```
replace-color image.png ff0000 0000ff
```
